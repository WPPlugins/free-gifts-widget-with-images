Free gifts and gift cards from gifteng.com

== Description ==

Free Gifts is a lightweight widget to display today\'s free gifts and gift cards on your wordpress blog or website. The widget is showing the latest gifteng images, title and description. Gifteng is a unique social community where you can give and receive things you love, for free.

== Installation ==

1. Upload `free_gifts directory` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the Free Gifts Widget and optionally set the border styles

== Frequently Asked Questions ==

Gifteng is a unique social community where you can give and receive things you love, for free. This widget will display the latest gifts posted today to your visitors trough your blog or website.
